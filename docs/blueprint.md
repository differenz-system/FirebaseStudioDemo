# **App Name**: TaskZen

## Core Features:

- Task List: Display tasks in a list format, sortable by due date, priority, or project.
- Project Management: Create and manage multiple projects to organize tasks.
- Due Dates and Reminders: Set deadlines and reminders for tasks.
- Task Prioritization: Prioritize tasks based on urgency and importance.
- AI Task Suggestions: AI tool that intelligently suggests due dates or categorizes tasks based on content.  It should only incorporate information or generate output that fits within a specific set of pre-programmed conditions.
- Smart Search and Sections: Search functionality to quickly find tasks, and group them into sections, like "today", "this week", and "upcoming".

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to promote a sense of calm and focus.
- Background color: Light gray (#F5F5F5) to provide a clean, distraction-free environment.
- Accent color: Muted green (#81C784) for task completion and positive reinforcement.
- Clear and modern font.
- Simple, geometric icons.
- Clean, minimalist layout with clear visual hierarchy.
- Subtle transitions and animations for feedback and engagement.
